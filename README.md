# Photo Collage Web App

A browser-based photo collager that can:

- Select images from a local folder.
- Choose from 10+ predefined collage styles (including one with rounded center image).
- Drag photos with the mouse inside collage frames to reposition crop and placement.
- Enable **swap mode** to drag one frame onto another and exchange their photo positions.
- Add a custom collage title.
- Apply special effects: Mosaic, Landscape Tone, Shadow 3D look, Tilted Photos, Vivid Colors, Vintage Film, Soft Glow, and Vignette Focus.
- Generate a collage on canvas and preview it instantly.
- Download the generated collage as PNG.

## Run

Open `index.html` in a browser.

## Notes

- Uses `webkitdirectory`/`directory` to let users pick folders.
- If a collage style needs more frames than selected photos, photos repeat automatically.
